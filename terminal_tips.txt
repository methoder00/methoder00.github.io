Linux Instruction

ls : list
cd : move
mkdir : make directory
rmdir : remove directory
rm : remove file
rm -rf : unconditional remove

Git Instruction

1. Connect with Git repository
    1) git clone address : clone exist repository, making directory which has same name
    2) git init : git start from existing directory
       git remote add origin(name) address
    3) disconnect : rm -rf .git

2. Update
    add > commit > push
                 < pull
    1) add : git add file               git st  : stage status
    2) commit : git commit file         git log : commit log
    3) git push origin master(branch)
    4) git pull origin master(branch)

3. Reset
    git reset ABC123 --hard     : 과거로 이동
    git revert ABC123           : 과거를 근 미래로 이동

3. Branch control
    1) git branch name : make branch
    2) git checkout name : change branch
    3) git merge name : merge target branch into current branch
    4) git rebase name : merge target branch into current branch (merge commit log)
    5) git branch -D name : delete branch
